# Flight-Price-Prediction
Flight Price Prediction using Machine Learning to estimate airfare based on airline, stops, duration, and travel date. Built with Python, Pandas, NumPy, and Scikit-learn. Multiple models tested, with Random Forest giving best accuracy. Includes data analysis, visualization, and insights into pricing trends.
